## 窗口设计

| Qt::MSWindowsFixedSizeDialogHint | 在Windows上为窗口提供精简的对话框边框。此样式传统上用于固定大小的对话框。 |
| -------------------------------- | ------------------------------------------------------------ |
| **Qt::FramelessWindowHint**      | 生成无边框窗口。用户无法通过窗口系统移动或调整无边框窗口的大小。 |
| **Qt::WindowStaysOnTopHint**     | 通知窗口系统窗口应该保持在所有其他窗口之上                   |
