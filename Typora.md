| 命令                                  | 解释                             |
| ------------------------------------- | -------------------------------- |
| `Ctrl+Shift+I`                        | 插入图片（可以直接拖动图片放入） |
| `Ctrl+T`                              | 插入表格                         |
| `Ctrl+回车`                           | 表格下方插入行                   |
| `Ctrl+Shift+删除键`                   | 表格删除行                       |
| `三个“-”或者“+“ 加回车(用减号较方便)` | 分割线                           |
|                                       |                                  |
